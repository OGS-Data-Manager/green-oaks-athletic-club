## Gotchas that cost real time today

- **Cloudflare login is `ezpeasydave@gmail.com`** — not `dave@ezpzps.com`. Wrong user
  gives "unauthorized access," which reads like a bad password. See
  `reference_cloudflare_account` memory.
- **Cloudflare Email Routing rules take an array of destinations**; the dashboard only
  exposes one picker. Use the API.
- **Cloudflare's Email Address Obfuscation** rewrites addresses to `[email protected]`,
  decoded by JS. Fine in a browser, **broken in an image/PDF export** — the flyer's
  contact line is wrapped in `<!--email_off-->`.
- **Stripe's payment cap lives on the Payment Link**, not the product in the catalog.
- Dave uses **Superwhisper**; odd words ("In the massive", "Orange") are transcription
  artifacts, not instructions. Ask.
- `index.html` is ~570 lines but huge in bytes (base64 fonts/images). **Never `Read` it
  whole** — it exceeds the token limit. Use targeted grep/python; lines 10, 11, 279 are
  blobs. **Use Edit, never Write.**
